📹 YouTube POSTer

Simple API for uploading videos to YouTube with API key authentication

Welcome to YouTube POSTer!

YouTube POSTer is a REST API service that simplifies uploading videos to YouTube. Instead of implementing OAuth flow in every client application, set up OAuth once on the server, then use simple API key authentication for all uploads.

Key Features

  • One-time OAuth 2.0 setup - authorize once, use forever
  • Simple API key authentication for clients
  • RESTful API with multipart file uploads
  • CORS enabled - call from any origin
  • Easy deployment to Render.com or any Node.js platform
  • Automatic token refresh - no re-authorization needed
  • Support for video metadata (title, description, tags, privacy)
  • Clean, well-documented codebase

Quick Start

1. Clone and install:

git clone <repository-url>
cd youtube-POSTer
npm install

2. Configure environment:

cp .env.example .env
# Edit .env with your Google OAuth credentials

3. Start the server:

npm start

4. Authorize with Google:

Visit http://localhost:3000/oauth/authorize

5. Upload a video:

curl -X POST http://localhost:3000/api/video \
  -H "X-API-Key: your-api-key" \
  -F "video=@video.mp4" \
  -F "title=My Video"
